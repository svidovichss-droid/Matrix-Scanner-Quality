# DataMatrix Scanner

**Мобильное приложение для анализа качества печати DataMatrix-кодов**  
Соответствие стандартам ГОСТ Р 57302-2016 / ISO/IEC 15415

---

## О приложении

DataMatrix Scanner — профессиональный инструмент для оценки качества печати двумерных штрихкодов формата DataMatrix непосредственно на производстве или в логистических цепочках.

Камера устройства захватывает изображение символа, система автоматически обнаруживает область DataMatrix, вычисляет комплекс метрологических показателей и формирует итоговую оценку качества по шкале A–F согласно международному стандарту ISO/IEC 15415 и его отечественному аналогу ГОСТ Р 57302-2016.

---

## Возможности

- **Автозахват** — автоматическое обнаружение и захват DataMatrix без ручного кадрирования
- **8 параметров качества** — полный набор метрик согласно стандарту
- **Шкала A–F** — международная четырёхбалльная система оценки (4.0 — отлично, 0 — неприемлемо)
- **Провал при отсутствии кода** — если DataMatrix не обнаружен в кадре, фиксируется результат F
- **История сканирований** — хранение, поиск и фильтрация всех предыдущих измерений
- **Офлайн-режим** — полная работоспособность без интернета
- **Экспорт результатов** — возможность поделиться протоколом измерения

## Параметры качества (8 метрик)

| Метрика | Полное название | Описание |
|---|---|---|
| **Decode** | Decode | Декодируемость — способность символа к надёжному декодированию |
| **SC** | Symbol Contrast | Символьный контраст между тёмными и светлыми модулями |
| **MOD** | Modulation | Модуляция — равномерность отражательной способности модулей |
| **MRD** | Min Reflectance Difference | Минимальная разница отражений между фоном и модулями |
| **AN** | Axial Non-uniformity | Осевая неравномерность шага модулей по осям X и Y |
| **UEC** | Unused Error Correction | Неиспользованный запас коррекции ошибок Reed-Solomon |
| **GN** | Grid Non-uniformity | Неравномерность позиционирования центров модулей |
| **FPD** | Fixed Pattern Damage | Повреждение стационарного шаблона (L-образная рамка) |

## Шкала оценок

| Оценка | Баллы | Значение |
|---|---|---|
| **A** | 3.5 – 4.0 | Отлично — соответствует всем требованиям стандарта |
| **B** | 2.5 – 3.4 | Хорошо — незначительные отклонения, код читается надёжно |
| **C** | 1.5 – 2.4 | Удовлетворительно — требует контроля и устранения дефектов |
| **D** | 0.5 – 1.4 | Плохо — серьёзные нарушения, считывание нестабильно |
| **F** | 0 – 0.4 | Неприемлемо — код не соответствует стандарту |

Итоговая оценка определяется по **минимальному** значению среди всех 8 параметров.

## Применяемые стандарты

- **ГОСТ Р 57302-2016** — Автоматическая идентификация. Кодирование штриховое. Спецификация символики DataMatrix
- **ISO/IEC 15415:2011** — Information technology — Automatic identification and data capture techniques — Bar code print quality test specification: Two-dimensional symbols

## Установка и разработка

```bash
# Установить зависимости
pnpm install

# Запустить Expo в режиме разработки
pnpm --filter @workspace/mobile run dev
```

Для запуска на устройстве используйте приложение **Expo Go** (iOS / Android) — отсканируйте QR-код из консоли.

- Node.js 18+
- pnpm 8+
